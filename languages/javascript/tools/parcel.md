# Parcel

## Introduction

* `parcel` is a `web application` bundler.

---

## Install

```bash
npm install -g parcel-bundler
yarn install -g parcel-bundler
```

--- 

## Example - Run

```
parcel index.html
```

--- 

## Example - watch

```
parcel watch index.html
```
