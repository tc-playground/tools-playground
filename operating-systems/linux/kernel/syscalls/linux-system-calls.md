# Linux System Call

* __fcntl__

    * The `fcntl` system call is the access point for several advanced `operations` on `file descriptors`. 

    * `fcntl(<file_descriptor>. <operation>)`

    * `fcntl` allows a program to place a `read lock` or a `write lock` on a file.
