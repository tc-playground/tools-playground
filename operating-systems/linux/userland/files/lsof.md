# lsof - List Open Files

## Introduction

* `lsof` - Lists `files` opened by `processes`.

* `lsof` file types include:
    * `regular file`, `directory`.
    * `block special file`,`character special file`.
    * `executing text reference`.
    * `library`.
    * `stream`.
    * `network file`.
        * `Internet socket`
        * `NFS file`.
        * `UNIX domain socket`
