# V8

## Introduction

* `V8` is Google's JavaScript engine.

* `V8` is written in `C` and `C++`.

* `V8` implements `ECMAScript` and `WebAssembly`.

* `V8` is used in `Chrome`, `Electron`, `NodeJS`, `Deno`, `Couchbase`, etc.
