# AlephJS

## Overview

* `Aleph.js` is a web framework for `Deno` inspired by `NextJS`.

* `Aleph.js` supports: 

    * Deno.
    * TypeScript.
    * ES module imports.
    * file-system routing.
    * SSR & SSG.
    * HMR with Fast Refresh.
    * No config needed.
