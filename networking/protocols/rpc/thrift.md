# Facebook Apache Thrift

## Introduction

* `Thrift` is an `interface definition language` and `binary communication protocol` for `RPC`.

* `Thrift` uses a `code generations engine`.

* `Thrift`  is similar to `Avro` and `Protocol Buffers`.
