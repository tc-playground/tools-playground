# Memory Map

```txt
+--------------+    0x0000 0000
| Flash ROM    |
|              |
+--------------|    0x2000 0000
| RAM          |
|              |
+--------------+    0x2000 7FFF
| I/O Ports    |
|              |
+--------------+
| Internal I/O |
|              |
+--------------+    0xE004 1FFF
```

> Addresses and sizes may vary.

* Memory is `byte addressable`.

* Memory starts at: `0x20000000`
