# Digital Ocean

* `Digital Ocean` is a cloud provider.

* `Digital Ocean` provides:

    * __Compute__

        * `Droplets` - Instances.

    * __Networking__

        * `Virtual PRivate Cloud (VPC)`

        * `DNS`

        * `Floating IP`

        * `Load Balancer`

        * `Firewall`
    
    * __Storage__

        * `Block Storage Volumes`
    
    * __Databases__

    * __Images__
    
    * __Containers__

        * `Kubernetes` - Container Orchestration.
    
    * __Monitoring__
l
* `Digital Ocean` tools:

    * `doctl` - CLI Tool.

    * `terraform` - Terraform provider.
